# Run as shiny app

Most of MSAexplorers features can also be accessed as a Shiny app.

## Installation

```bash
pip install shiny
git clone https://github.com/jonas-fuchs/MSAexplorer
pip install .
cd MSAexplorer
shiny run app
```

This will run the shiny app which can be accessed in the browser.
